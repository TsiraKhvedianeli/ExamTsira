рЃћрЃА рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў Android рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф Kotlin рЃћрЃюрЃљрЃќрЃћрЃљ рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў, рЃљрЃЦ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃюрЃљрЃ«рЃЮрЃЌ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃљрЃарЃбрЃўрЃАрЃбрЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃЮрЃърЃБрЃџрЃљрЃБрЃџрЃў рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљ.
рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃўрЃАрЃљрЃА рЃњрЃЋрЃ«рЃЋрЃЊрЃћрЃЉрЃљ рЃЏрЃ«рЃљрЃбрЃЋрЃарЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃљрЃџрЃў рЃАрЃљрЃЊрЃљрЃф рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃЌрЃў: рЃцрЃЮрЃбрЃЮ, рЃАрЃљрЃ«рЃћрЃџрЃў, рЃњрЃЋрЃљрЃарЃў, рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃгрЃћрЃџрЃў, рЃњрЃљрЃарЃЊрЃљрЃфрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃгрЃћрЃџрЃў.
рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃљрЃљрЃарЃЕрЃўрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў, рЃЊрЃљ рЃЊрЃљрЃГрЃћрЃарЃўрЃА рЃЊрЃарЃЮрЃА рЃњрЃљрЃЊрЃљрЃЋрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ рЃАрЃљрЃЊрЃљрЃф рЃЕрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџрЃўрЃљ рЃЏрЃљрЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў 10 рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃљрЃарЃў рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃгрЃџрЃћрЃЉрЃў.
рЃърЃўрЃарЃЋрЃћрЃџ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃљрЃбрЃЋрЃарЃўрЃА рЃгрЃљрЃерЃџрЃљ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ рЃљрЃю рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЋ рЃњрЃљрЃгрЃћрЃЋрЃЌ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃЮрЃўрЃерЃџрЃћрЃЉрЃљ рЃАрЃўрЃўрЃЊрЃљрЃю.

­ЪЊ▒ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- рЃЏрЃ«рЃљрЃбрЃЋрЃарЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ Retrofit-рЃўрЃЌ.
- рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃ«рЃљрЃбрЃЋрЃарЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ:
  - рЃАрЃљрЃ«рЃћрЃџрЃў
  - рЃърЃћрЃарЃўрЃЮрЃЊрЃў
  - рЃАрЃБрЃарЃљрЃЌрЃў (рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА Picasso-рЃўрЃА рЃџрЃЮрЃЊрЃўрЃюрЃњрЃА)
- рЃЊрЃљрЃљрЃгрЃЎрЃљрЃърЃБрЃюрЃћрЃЌ рЃЏрЃ«рЃљрЃбрЃЋрЃљрЃарЃќрЃћ рЃЊрЃљ рЃњрЃљрЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ рЃЏрЃћрЃЮрЃарЃћ рЃћрЃЎрЃарЃљрЃюрЃў, рЃАрЃљрЃЊрЃљрЃф рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЏрЃ«рЃљрЃбрЃЋрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃў рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃгрЃџрЃћрЃЉрЃў.
- рЃАрЃЋрЃљрЃўрЃърЃўрЃЌ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃгрЃљрЃерЃљрЃџрЃЮрЃЌ рЃЏрЃ«рЃљрЃбрЃЋрЃарЃћрЃЉрЃў рЃАрЃўрЃўрЃЊрЃљрЃю.
